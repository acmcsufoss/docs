# projects

List of initiatives owned by the Open Source Software team.

## Development

Make sure to install Deno:
<https://deno.land/manual/getting_started/installation>.

Build the project:

```sh
deno task build
```

Build the project in development mode:

```sh
deno task dev
```

Serve the static site:

```sh
deno task serve
```

Format the project:

```sh
deno fmt
```

Lint the project:

```sh
deno lint
```

Update dependencies:

```sh
deno task udd
```
